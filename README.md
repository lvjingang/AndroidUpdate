# AndroidBeatUpdate
制作一个Android最简洁的更新组件
感觉GitHub上的更新太臃肿，很多主题包，还有很多其实根本用不上的功能，导致体积很大.于是自己写了一个
## 如何导入？

* Android Studio

          ```
          allprojects {
            repositories {
              ...
              maven { url 'https://jitpack.io' }
            }
          }
          ```
  
          ```
          compile 'com.github.shuhaoSCode:AndroidBeatUpdate:-SNAPSHOT'
          ```
  
* eclipse。。。请自行copy class。
